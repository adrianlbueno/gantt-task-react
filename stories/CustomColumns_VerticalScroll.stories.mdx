# Simple with only the Title column

By passing a columns collection with only the `TitleColumn`

```jsx
const tasks = useMemo(() => initTasks());
const columns = [{
          Cell: TitleColumn,
          width: 210,
          title: "Name",
          id: "Name",
        }];
return <Gantt
          columns={columns}
          tasks={tasks} />;
```

you get the following
